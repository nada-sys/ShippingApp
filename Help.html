Map generation tool

After initial installation please ensure you set the MyPC Servername via the settings tab and the
desired refresh rate of the map(s). Please test the connection to ensure that there are no
connectivity issues.

If all is well please follow the steps below to make a floor plan
  • 1. Click New
  • 2. Select the Locations that you would like to show on the map
  • 3. Create your floor plan either by using the supplied drawing tools or by importing an existing
    floorpan.You can import a jpg/png/bmp file
  • 4. Drag and drop the PC’s from the left pane to the desired position on your plan.
  • 5. Once all the PC’s are in position click Save and provide a file name.
  • 6. Preview will show the Map as it would appear.

Modify existing Plan

  • 1. Click Open
  • 2. Browse to the Plan you wish to modify
  • 3. The plan will open and you can modify
  • 4. Click Save to finalise the changes.

Viewing Map/Plan on large screen

The map/plan is designed to be shown on a PC or plasma screen and will scale to the browser size
available on the display. To view the Map you would browse to the following URL:

http://servername/MapWebApp/showmap.aspx

(Please replace the serve name with you r server name)

This will display a list of all available maps if you click one of the items in the list the map will show and generate a URL like:

http://servername/MapWebApp/GenerateMap.aspx?mac.xml&30
